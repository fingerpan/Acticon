# [vue-acticon](http://www.acticon.top)

> an vue.js move icon  

## Build Setup

``` bash
npm install vue-acticon --save
```

## example:

```

<template>
    <div>
        <music-round></music-round>
    </div>
</templage>

<script>
  ...
  import { musicRound } from "vue-acticon"
  ...
  
  export default{
      data() {},
      components:{
        musicRound
      }
  }
  
</script>

```
